# SkillForge - Industry-Oriented Guided Freelancing Platform

A modern, enterprise-level EdTech platform that connects students, mentors, faculty, and clients for guided skill development and real-world freelancing experience.

## 🎯 Platform Overview

SkillForge is a comprehensive skill incubation platform designed to help students transition from learning to industry-ready professionals through structured mentorship, monitored assessments, and real-world project experience.

## ✨ Key Features

- **Multi-Role Dashboards**: Separate interfaces for Students, Mentors, Faculty, Clients, and Administrators
- **Skill Level System**: Progressive certification (Level A/B/C/D) based on monitored exams
- **Guided Learning**: AI-powered course recommendations and structured learning paths
- **Real-World Projects**: Students work on actual client projects under mentor supervision
- **Online Exam System**: Fullscreen proctored exams with webcam and audio monitoring
- **Project Management**: Kanban-style boards for tracking project progress
- **Analytics Dashboard**: Comprehensive metrics and insights for platform performance
- **Dark/Light Mode**: Built-in theme support for user preference

## 🚀 Quick Start

### Available Routes

- `/` - Landing page with platform overview
- `/student` - Student dashboard with learning progress
- `/mentor` - Mentor dashboard for student monitoring
- `/faculty` - Faculty dashboard for administrative controls
- `/client` - Client dashboard for project management
- `/exam` - Online examination interface
- `/learning` - Learning portal with courses and paths
- `/projects` - Project management with Kanban board
- `/analytics` - Analytics and reports dashboard
- `/admin` - Admin panel for user management

### Navigation Tips

1. Start at the landing page (`/`) to see the platform overview
2. Use the navbar links to quickly access different dashboards
3. Each dashboard has realistic dummy data and interactive components
4. Toggle between dark and light modes using the theme switcher
